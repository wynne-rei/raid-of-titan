# RAID OF TITAN — ゲーム仕様書

**最終更新:** 2026-04-08  
**エンジン:** Three.js r160 (WebGL ブラウザゲーム)  
**ジャンル:** リアルタイムストラテジー / タワーディフェンス  
**コンセプト:** 終わらない三国戦争 — 3国がGDPを巡って争う無限バトル

---

## 1. ゲームモード

| モード | 説明 |
|--------|------|
| **プレイヤーモード** | 1人のプレイヤー + 2体のAI bot（各国1体）で3国対戦 |
| **ダイバーモード** | 観戦者として参加し、ギフトカードで配信者を支援 |
| **Co-opモード** | 最大4人が同じ国で協力プレイ（WebRTC/PeerJS） |
| **チュートリアル** | 3ステップの初心者ガイド（移動→宝箱→戦闘） |

## 5. GDPシステム（資源）

GDPは国の資源。ユニットの配置や施設の建設に消費する。

### GDP獲得源

| 源 | 量 |
|----|-----|
| 基本パッシブ | 20 GDP/秒 |
| 存命ユニット1体あたり | 0.2 GDP/秒 |
| 存命タワー1つあたり | 1 GDP/秒 |
| 農場（施設） | 基本パッシブ +15% |
| 祭壇（施設） | GDP全体 +30% |
| 建物/施設チェスト | +200 GDP |
| レイドボス撃破 | +300 GDP |

### GDP消費

| 用途 | コスト |
|------|--------|
| カード配置 | カードごとに40〜300 GDP |
| ダイバーギフト | 100 GDP/回 |

### 領土拡張（GDPベース）

GDP累計に応じて領土が広がる。閾値: 500 → 1200 → 2500 → 5000

## 7. レイドボスシステム

定期的にフィールドに出現する強力なボス。3種類で各々固有の特殊能力を持つ。

| ボス | 二つ名 | アイコン | 能力 | 行動 |
|------|--------|---------|------|------|
| ゴーレム | 滅城のゴーレム | 🗿 | destroyStructures | 敵施設を3つ破壊して去る |
| 魔女 | 魅了の魔女 | 🧙‍♀️ | convertTroops | 敵全兵士を自国に寝返らせて去る |
| 氷龍 | 氷結の龍 | 🐲 | freezeAll | フィールド全体を5秒間凍結して去る |

| 定数 | 値 |
|------|-----|
| 出現間隔 | 30〜45秒（ランダム） |
| 予告時間 | 10秒 |
| 撃破報酬 | GDP+300, XP+50（撃破者）, XP+25（味方） |

## 11. 戦闘メカニクス

### ターゲティング
1. 最も近い有効な敵（ユニットまたはタワー）
2. 射程内
3. 対象タイプが一致（地上/空中/両方/建物）
4. 同盟ではない

### ダメージ
- **射程2超:** 弾丸を発射 → 着弾でダメージ
- **射程2以下:** 近接直接ダメージ
- **スプラッシュ:** 着弾点周囲のユニットにも波及

### 状態異常
| 効果 | 持続時間 | 内容 |
|------|----------|------|
| 凍結（Freeze） | 3〜5秒 | 移動・攻撃不可 |
| レイジ（Rage） | 6秒 | 速度/攻撃力 +40% |

## 12. AIボット

各国にAI botが1体ずつ配置される。

| パラメータ | 値 |
|-----------|-----|
| ユニット配置間隔 | 20秒（±5秒のブレ） |
| 配置ロジック | 国固有カードからランダム選択 |
| 配置条件 | GDP ≥ カードコスト |
| 配置場所 | 自国タワー周辺（±100ユニット） |
| 施設建設 | 数分おきに壁/塔を自動配置 |

## 14. 図鑑（Encyclopedia）

タイトル画面の「📖 図鑑」から閲覧可能。

- **タブ:** 国別（火/氷/土）、キャラクター、ユニット
- **内容:** 全カードのアイコン・名前・コスト・ステータス・説明
- **フィルター:** タイプ別（兵士/呪文/建物/施設）

## 15. セーブ/ロード

- **自動保存:** 30秒ごと
- **保存先:** ブラウザのlocalStorage
- **保存内容:** ゲーム状態、エンティティ、タワー、施設、GDP、レベル、デッキ等
- **復元:** 「プレイヤーとして戦う」→「続きから始める」
